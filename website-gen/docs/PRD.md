## 0. 핵심 결정 요약
- **핵심 서비스**: 정보+도구 제공형 (유료 상담 아님)
- **전환 목표**: 인스타그램(두잉두잇) 팔로워/커뮤니티 확보
- **타깃**: 전체 연령, 검진수치 처음 높게 나온 사람 (넓게)
- **차별점 1순위**: 습관 코칭 프레임 (행동변화단계, TTM 기반) — 13장 도구로 구체화, 홈페이지 빌드 후 추가 예정
- **빌드 순서**: 홈페이지(현재 구조) 먼저 완성 → 습관단계 자가진단 도구 + 인사이트 블로그는 추후 추가. 홈페이지에는 두 기능의 **"준비 중" 티저**를 미리 배치해 둔다.
